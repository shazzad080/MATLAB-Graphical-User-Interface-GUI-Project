## What's This All About?

Ever struggled to visualize how logic gates work or what digital modulation looks like? I built this app to solve exactly that problem. It's a hands-on tool where you can click buttons, see truth tables pop up instantly, watch waveforms dance across your screen, and actually *understand* what's happening under the hood.

Think of it as your personal digital electronics playground, built right in MATLAB.

## Getting Started (It's Easy!)

### What You'll Need
- MATLAB installed on your computer (anything from R2019a onwards works great)
- That's pretty much it!

### Setting It Up
1. Grab the code from this repo:
   ```bash
   git clone https://github.com/yourusername/digital-logic-modulation-simulator.git
   ```
2. Fire up MATLAB
3. Navigate to where you saved the files
4. Open the `.mlapp` file in App Designer
5. Hit that run button and you're good to go!

## How to Use It

### Logic Gates Are Your Friends
1. Click on the **Digital Logic Design** tab
2. Pick any gate that catches your eye (maybe start with AND?)
3. Watch the truth table generate itself
4. Check out those sweet waveforms showing inputs and outputs

### Modulation Station (Coming Soon!)
1. Head over to the **Digital Modulation** tab
2. Choose a modulation type (ASK is pretty cool for beginners)
3. Type in some binary numbers (like 101010)
4. Adjust the knobs and sliders
5. Marvel at the diagrams that pop up

## Quick Logic Gate Cheat Sheet

Here's what each gate does in plain English:

| Gate | What It Does |
|------|--------------|
| **AND** | Only says "yes" when EVERYONE says "yes" |
| **OR** | Says "yes" if ANYONE says "yes" |
| **NOT** | The rebel - always does the opposite |
| **NAND** | AND gate having a bad day (opposite of AND) |
| **NOR** | OR gate's evil twin (opposite of OR) |
| **XOR** | Only happy when inputs disagree |
| **XNOR** | Only happy when inputs match |
